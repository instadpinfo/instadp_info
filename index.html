Method 1: Using the Chrome Browser Extension "DownAlbum"

  1. Install the DownAlbum extension from the Chrome Web Store.
  2. Navigate to the Instagram page where the image you want to download is located
  3. Click on the DownAlbum icon in the upper right corner of the screen (it looks like a down arrow)
  4. A new window will open showing all of the images on the page. Select the image you want to download, then click on the "Download" button
  5. The image will be saved to your computer in the default location chosen by your browser (usually the "Downloads" folder)
